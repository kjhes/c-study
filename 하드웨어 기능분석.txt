하드웨어 기능 분석하기

/하드웨어 시스템
각각의 기능을 가능캐 하는 구성요소가 조합되어 
복잡한 작업을 수행하는 하드웨어 장치 임베디드 시스템 이라 불리기도 함
하드웨어 시스템은 각 기능을 가능캐 하는 하드웨어들의 모음
-하드웨어 구성요소 모델 
하드웨어는 단순한 제품이 아니라
시간(정보 처리 박자) + 에너지(전원) + 규칙(프로토콜) + 통로(버스) 등 여러 부품위에서 동작하는 회로들의 집합이다

/하드웨어 구성요소
마이크로프로세서 or 마이크로컨트롤러:cpu로서 기계어 코드를 계산하는 회로를 말함 임베디드 시스템에서는 mcu라는 cpu(마이크로프로세서) + 주변장치(입출력 기능 포함)가 한 칩에 들어간 다기능 장치로서 사용된다

/메모리
반도체들로 만들어진 정보를 기억하는 기억 소자
마이크로프로세서(cpu)실행을 위한 프로그램 , 데이터 저장의 역할을 수행
전원이 없을때 저장된 데이터의 잔류 여부에 따라 휘발성 또는 비휘발성 메모리등으로 구분한다

//휘발성 메모리:메모리속 정보를 유지하기 위해선 전원이 필요한 기억소자 데이터의 임시 저장에 사용되며 ram이라 불린다

/SRAM(Static RAM)
RAM에 전원이 공급되는 한 그 내용이 보존됨 주소와 관계없이 소요되는 시간이 일정한데 이 이유는 일반적인 디스크 형태 원반형이 아니라 배열처럼 되어져 있어서 
주소로 접근하면 바로 바로 접근이 되기 때문에 데이터를 읽는데 걸리는 시간이 일정하다 빠르고 일정한 접근을 만들기 위해 더 많은 회로를 쓰기 때문에 sram 이 비싸다
마이컴(마이크로컴퓨터) 같은 구성에서 자주 활용된다
마이컴은 CPU 메모리 입출력이 하나로 붙여져 만들어진 미니 컴퓨터
동기식 SRAM: 클록 신호를 기준으로 일정한 리듬에 맞쳐 동작 - 필요한 이유 CPU가 기본적으로 클럭에 맞춰서 동작하기 때문에 
비동기식 SRAM: 클록 신호 없이 그냥 신호가 들어오면 바로 동작 - 지연이 상황마다 조금씩 다르기 때문에

/DRAM(Dynamic RAM)
가만히 두면 점점 전하가 새어 나가서 계속 재 충전을 해 줘야지 데이터를 보관할 수 있음
축전소자(커패시터:전기를 잠깐 저장하는 부품)로 구성되어 있음 
SRAM에 비해 구조가 단순하고 용량이 높고 싼 대신 속도가 많이 느림
싼맛에 그냥 리프레쉬 비용을 감수하고 사용 리프레쉬 방법에 따라 DRAM SDRAM DDR-SDRAM 등으로 분류된다.

//비휘발성 메모리:전원이 공급되지 않아도 정보를 유지하는 저장 장치

/플래쉬 메모리
주소 지정 방식에 따라 NAND , NOR 형식으로 구분된다 NAND 방식은 직접 접근 가능 해서 CPU가 바로 코드를 실행 할 수 있다
NOR Flash:직접 접근 가능 CPU가 빠르게 읽고 실행 가능 - 아무 페이지부터 읽어서 처리하는 구조 펌웨어에서 주로 사용됨
NAND Flash:블록단위로 읽고 씀 저장장치에서 주로 사용됨

/저장 장치
프로그램이 돌아갈때 cpu가 사용하지 않는 디스크로 
메모리 소자들에 비해 느리지만 싸고 대용량의 정보를 
블럭단위로 데이터를 교환하고 하드디스크 메모리 카드 테이프 리코더 등으로 분류된다

/입출력 장치
정보를 주고 받는 통로 역할을 하는 장치
데이터를 주고 받는 형식에 따라 직렬 장치와 병렬 장치로 구분된다
또 동기식이나 비동기식으로 구분된다

/전원 장치
전력을 공급하는 장치 컴퓨터는 12V 5V 3.3V 1.8V등
여러가지 종류의 직류 전압을 필요로 하기에 우리가 가정용으로 사용하는
220V의 교류 전기를 직류 전압으로 변환하여 공급하는 역할을 한다
-과전류 감지 회로 : 쇼트가 발생하면 큰 전압이 순간적으로 폭발하듯 가해져 부품이 손상될 수 있기 때문에 이를 막기 위한 회로가 필요하다.감지 회로의 저항기의 위치에 따라 하이사이드 전류 감지 회로와 로우사이드 전류 감지 회로로 구분할 수 있다
하이사이드 전류 감지 : 쇼트를 감지하는 저항기가 전원과 부품들 사이에 위치하며 대부분의 전류 감지 회로에서 사용된다.
